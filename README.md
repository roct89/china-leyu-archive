# China Leyu Archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容以中文为主，涉及不同主题，不针对任何单个域名或具体网站。

## 项目简介

本项目是一个静态 HTML 页面的归档集合。页面内容由贡献者整理、编写或收集，并以独立文件形式存放于仓库中。所有页面均为纯前端实现，无需后端服务即可在浏览器中直接打开查看。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
│   ├── example/    # 示例页面文件夹（可删除或替换）
│   └── ...
├── assets/         # 公共资源文件（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：所有归档的 HTML 页面均放置于此目录下，建议按主题或时间分类存放。
- `assets/`：存放页面共用的资源文件，如 CSS、JavaScript、图片等。
- `index.html`：可选的入口页面，用于展示页面列表或导航。

## 页面归档说明

- 每个 HTML 页面应保持独立，尽量不依赖外部资源（除 `assets/` 目录下的公共文件）。
- 页面内容需符合法律法规，不包含恶意代码、钓鱼信息或侵犯他人权益的内容。
- 归档的页面可能来自不同来源，版权归原作者所有。本仓库仅作为存档和展示用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可以通过 `index.html` 浏览页面列表（如果存在）。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有页面。
- 请确保新增页面不包含任何具体域名、网址、商业推广或诱导点击内容。
- 如果发现页面存在违规或错误，请通过 Issue 反馈，我们会及时处理。
- 本仓库不保证所有页面长期可用或内容更新，仅以“现状”提供。
